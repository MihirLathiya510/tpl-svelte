# tpl-svelte

A modern full-stack Svelte template with PostgreSQL and Drizzle ORM.

## Quick Overview

```bash
npm create svelte@latest my-app -- --template https://github.com/MihirLathiya510/tpl-svelte
cd my-app
npm install
npm run dev
```
